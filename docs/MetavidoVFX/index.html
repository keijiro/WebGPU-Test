MetavidoVFX - Unity WebGPU VFX Graph Sample

MetavidoVFX is a Unity WebGPU sample project that plays back a Metavido video clip recorded with iPhone 14 Pro Max, visualizing the scene with particle systems.

This sample uses the H.264 decoder by default. For Chromium, try this link to use the VP9 decoder instead.

Explore the source project on GitHub here.
